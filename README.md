# Student Registration System

## Description
A web-based student registration system built using HTML, CSS, and JavaScript.
It allows users to add, edit, delete student records with localStorage support.

## Features
- Add / Edit / Delete students
- Data saved using localStorage
- Responsive design (Mobile / Tablet / Desktop)
- Input validation

## Technologies Used
- HTML
- CSS
- JavaScript
